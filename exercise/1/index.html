Exercise 1: The Navigation Bar

Hi, Welcome to the first exercise. Your task is to style a navigation bar which has an nested list. This topic focusing on list styling and css selectors. BTW, before you start coding let me introduce your tools.

On your right. You should see an editor. This is a tool which you will use to code a CSS code.

On the top of window, you should see triple dots at the center on both editor and this window. You can grap this and drag around where ever you like :>

Also on the top-right corner of the editor, as you can see you can give up and request a solution.

But remember your appoach may differ from the solution since it has a lot of way to solve the exercise, so don't be stress if your solution doesn't be the same.

Lastly on the bottom-right corner, you can resize the window to fit your workspace.

GLHF.

Rules

  1. You are free to use browser debugging tools to inspect elements in the page.
  2. You can use all css property and css selectors in this exercise.
  3. Don't use an external code editor. We understand that without an auto-suggestion is quite inconvenient, but this is an essential step not only to CSS, but to the world of programming.
  4. Don't use javascript. This is a CSS practice not the javascript.
